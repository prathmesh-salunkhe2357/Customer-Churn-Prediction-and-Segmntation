# Customer Churn Prediction and Customer Segmentation

## Overview

This project is an end-to-end Machine Learning solution that predicts customer churn and segments customers into meaningful groups using clustering techniques. The project demonstrates the complete ML workflow, from data preprocessing and exploratory data analysis to model training, evaluation, and business insights.

## Problem Statement

Customer churn is a major challenge for businesses because losing existing customers directly affects revenue. The objective of this project is to:

* Predict whether a customer is likely to churn.
* Segment customers based on their characteristics.
* Generate actionable business insights to improve customer retention.
